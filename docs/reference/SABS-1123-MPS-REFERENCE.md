## Column Definitions

| Column | Description |
|--------|-------------|
| NB | Nominal Bore (mm) |
| Pressure | Pressure class designation (kPa/type) |
| D | Flange outside diameter (mm) |
| b | Flange thickness (mm) |
| d4 | Raised face diameter (mm) |
| f | Raised face height (mm) |
| Holes | Number of bolt holes |
| d1 | Bolt hole diameter (mm) |
| Bolt | Bolt size |
| PCD | Pitch circle diameter (mm) |
| Mass | Flange mass (kg) |

## BS4504 Special Flanges (Large Sizes)

**Note:** SABS 1123 only covers sizes up to 600NB. For larger sizes (700NB, 800NB, 900NB, 1000NB, 1200NB), use BS4504 special flanges as per MPS Technical Manual pages 115-120.

These special flanges are "consistent with but not covered by" BS4504 Part 1:1969 standard.

### Available Special Flange Sizes

| NB | Pipe OD | Pressure Classes Available |
|----|---------|---------------------------|
| 700 | 711.2mm | PN10, PN16, PN25, PN40 |
| 800 | 812.8mm | PN10, PN16, PN25, PN40 |
| 900 | 914.4mm | PN10, PN16, PN25, PN40 |
| 1000 | 1016.0mm | PN10, PN16, PN25, PN40 |
| 1200 | 1219.2mm | PN10, PN16, PN25, PN40 |

### Flange Types Added

- /3 (Slip-On)
- /8 (Blind)

Migration file: `1776200000000-AddM38BoltAndBs4504SpecialFlanges.ts`

## Notes

1. **SABS 1123 Coverage**: Standard sizes 15NB to 600NB only
2. **Special Flanges**: For sizes > 600NB, use BS4504 special flanges
3. **Mining Sizes**: The MPS manual uses special mining nominal sizes that may not directly correspond to standard NB. The migration uses standard pipe NB based on pipe OD values
4. **Mass Estimates**: Mass values for special flanges are calculated estimates based on dimensions
