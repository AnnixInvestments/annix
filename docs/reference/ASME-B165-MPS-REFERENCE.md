## Column Definitions

| Column | MPS Label | Description |
|--------|-----------|-------------|
| NPS | Nominal Pipe Size | Pipe size in inches |
| NB | - | Nominal Bore in mm (metric equivalent) |
| D | D | Flange outside diameter |
| b | t | Flange thickness |
| d4 | G | Raised face diameter |
| f | - | Raised face height (0.06" for Class 150-600, 0.25" for Class 900+) |
| Holes | - | Number of bolt holes |
| d1 | - | Bolt hole diameter |
| Bolt | - | Bolt size |
| PCD | C | Pitch circle diameter (bolt circle) |

## Discrepancy Corrected

### Issue Found
The original ASME B16.5 migration (1770300000000-AddComprehensiveAsmeB165FlangeData.ts) used **pipe OD values** for the d4 (raised face diameter) column instead of the actual **raised face diameter (G)** values from the standard.

### Correction Applied
Migration `1776300000000-CorrectAsmeB165RaisedFaceDiameters.ts` updated 289 flange dimension records with correct d4 values.

### Example of Correction (150NB Class 150)
| Field | Before (Incorrect) | After (Correct) |
|-------|-------------------|-----------------|
| d4 | 170 (pipe OD) | 216 (raised face dia) |

## Notes

1. **Imperial to Metric Conversion:** Values converted using 25.4mm per inch
2. **Flange Types Available:** /2 (Weld Neck), /3 (Slip-On), /8 (Blind)
3. **Blind Flanges:** d4=0 because they don't have a raised face bore opening
4. **Pressure Classes:** 150, 300, 400, 600, 900, 1500, 2500
5. **Bolt Designations:** Imperial sizes (1/2", 5/8", 3/4", 7/8", 1", 1-1/8", etc.)
