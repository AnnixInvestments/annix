## 12. ISO Size Type Codes

### 12.1 Code Structure

The ISO size type code consists of 4 characters:
- **1st character**: Length code (2=20', 4=40', L=45')
- **2nd character**: Height/Width code (2=8'6", 5=9'6" HC)
- **3rd character**: Type code (G=General, R=Reefer, U=Open/Hardtop, P=Flat, T=Tank, V=Ventilated)
- **4th character**: Subtype code

## 13. Loading Guidelines

### 13.1 Weight Distribution

- Distribute cargo weight evenly across the floor
- Centre of gravity should be as low as possible
- Heavy items should be placed at the bottom
- Concentrated loads must use dunnage for distribution

### 13.2 Stacking Considerations

| Container Type | Max Stack Weight (at sea) |
|----------------|--------------------------|
| 20' General Purpose | 192,000 kg (6 high) |
| 40' General Purpose | 192,000 kg (6 high) |
| 40' High Cube | 192,000 kg (6 high) |
| Refrigerated | Varies by unit |

### 13.3 Door Capacity

- Standard containers have 2 doors (left and right)
- Each door can support approximately 2,700 kg when closed
- Lashing rings inside container rated at 1,000-2,000 kg
