## What this is for

Compliance tracks the documents a candidate needs — ID, qualifications, certificates, licences, clearances — and whether each is missing, received, verified, expiring or expired.

## Step-by-step

1. Open **Compliance** and select **Add document**.
2. Choose the candidate and the document type (or "Other" for a custom name).
3. Set the status and, where relevant, the expiry date.
4. Use the status filters at the top to see what's **missing** or **expiring** across all candidates.

## Rules

- Documents are listed agency-wide so anyone can see what's outstanding.

## Tips

- Filter to *Expiring* and *Expired* regularly so a lapsed licence never derails a placement.
