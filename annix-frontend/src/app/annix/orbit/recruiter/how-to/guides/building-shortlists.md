## What this is for

A shortlist packages your best candidates for one client vacancy and tracks where it is — draft, sent, reviewed.

## Step-by-step

1. Open **Shortlists** and select **Create shortlist**.
2. Name it (e.g. "Sales Manager — March"), set the job title and link the client.
3. Tick the candidates to include.
4. Add notes for the client explaining why these candidates fit.
5. Set the status to **Ready to send**, then **Sent** once it's with the client.

## Rules

- Marking a shortlist **Sent** records a share event against each candidate's Activity trail (POPIA).
- Only submit candidates who have consented to share.

## Tips

- Keep one shortlist per vacancy so the status reflects that specific client conversation.
